# Ticket Scout

**TicketScout** is a Node.js web application, powered by Express, designed to help small businesses track and manage tickets, payments, and customers. This streamlined tool allows service providers to efficiently manage their customer interactions and financial transactions, making it ideal for businesses that rely on client services.

**View Ticket Scout:** [www.ticketscout.render.com](https://ticketscout-ji8f.onrender.com)

## Motivation

After gaining 2 years of experience as a computer technician and working with various software tools to track repairs, I recognized the need for a simpler yet effective solution. My goal was to develop a user-friendly web application that small businesses, particularly repair shops, could easily adopt to manage their operations with greater efficiency and ease.

## Features

### Core Operations

- **Comprehensive Ticket Management**: Easily create, update, and track repair tickets for various electronic devices.
- **Customer Relationship Management (CRM)**: Store customer details, track service history, and maintain detailed records of interactions.
- **Seamless Payment Tracking**: Monitor payments with the ability to track completed and pending transactions.

### Advanced Features

- **Real-Time Shipment Tracking**: Keep customers informed with real-time updates on repair shipments.
- **Automated SMS Notifications**: Send SMS updates to customers about repair progress, payment reminders, and pickup notifications.

### User and Access Management

- **Admin and Employee Permissions**: Configure different access levels for admins and employees to manage tasks and view sensitive information according to their roles.
- **User Time-Clock Tracking**: Track employee work hours and manage time logs efficiently, helping with payroll and scheduling.

- **Intuitive Dashboard**: Manage daily operations with an easy-to-navigate dashboard that offers quick access to all key areas.

## Technologies

**Client:** HTML, SCSS, Javascript

**Server:** Node, Express

**Database:** MongoDB

**API Integration:** Pusher web sockets, Google maps, Google geolocation, Twilio SMS api, GoShippo tracking api
